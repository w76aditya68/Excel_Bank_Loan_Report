## 📋 Problem Statement

Banks need a reliable way to monitor lending activities, assess portfolio health, and track month-over-month performance. Without this visibility, risks go undetected and growth opportunities are missed.

This project builds a **two-dashboard Excel report** that:
- Tracks KPIs  
- Separates good and bad loans  
- Provides borrower insights

## 💡 Key Insights

- **Strong Portfolio Health:**  
  86.2% of loans are performing, but 13.8% accounts for $65.5M, indicating notable credit risk.

- **High Loss on Bad Loans:**  
  $37.3M recovered from $65.5M → ~43% loss gap, showing weak recovery efficiency.

- **Consistent Growth Trend:**  
  Applications peak in December (4,314), highlighting seasonal demand opportunities.

- **Improved Performance Metrics:**  
  Funded (+13%) and received (+15.8%) growing faster than applications → better conversion and repayment.

- **Geographic Risk Concentration:**  
  California contributes ~18% of applications → dependency on a single region.

- **Rising Risk Indicators:**  
  Increase in interest rate and DTI suggests lending to higher-risk borrowers.

## 🛠️ Tools & Skills Used

## 🚀 How to Use

1. Download: `Bank_Loan_Report.xlsx`  
2. Open in Excel (2016 or later recommended)  
3. Navigate:
   - Summary Dashboard → KPIs & loan health  
   - Overview Dashboard → trends & insights  
4. Use slicers to filter:
   - Loan Status  
   - Loan Term  
   - Loan Purpose
